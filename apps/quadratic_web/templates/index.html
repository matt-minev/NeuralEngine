Performance Analysis

📊 Performance Metrics Radar

📖 Understanding Performance Metrics

📈
R² Score (Coefficient of Determination)

Measures how well the model explains the variance in the target data. Range: 0 to 1.

  • 1.0: Perfect predictions
  • 0.8+: Excellent performance
  • 0.6-0.8: Good performance
  • <0.6: Needs improvement
📉
MSE (inv) - Inverted Mean Squared Error

Shows accuracy by inverting MSE values. Higher values indicate better performance.

  • MSE measures average squared differences between predicted and actual values
  • Inverted for easy comparison (higher = better)
  • Normalized by maximum MSE in dataset
🎯
MAE (inv) - Inverted Mean Absolute Error

Average absolute difference between predictions and actual values (inverted).

  • Less sensitive to outliers than MSE
  • Represents average prediction error magnitude
  • Inverted and normalized for better visualization
🏆
Accuracy (10% Tolerance)

Percentage of predictions within 10% of actual values.

  • 90%+: Exceptional accuracy
  • 80-90%: Very good accuracy
  • 70-80%: Acceptable accuracy
  • <70%: Requires optimization
